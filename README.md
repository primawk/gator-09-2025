gator is a rss feed cli tool that stores feeds and posts that you are interested in.
users can create different account to login and have preferred contents with follow and unfollow commands.

List of commands:
npm run start login 
npm run start register
npm run start reset
npm run start users
npm run start agg
npm run start addfeed
npm run start feeds
npm run start follow
npm run start following
npm run start unfollow
npm run start browse
